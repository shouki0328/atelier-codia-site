# Atelier Codia ユースケースシナリオ

## ステークホルダー

| ステークホルダー | 目的 |
|---|---|
| 訪問者 | 世界観、代表作品、素材感を短時間で把握する |
| ブランド閲覧者 | 作品写真、素材感、制作の空気を静かに眺める |
| ブランド所有者 | 小さな個人ブランドとして、高級感のある見え方で公開判断する |
| 実装/運用者 | 静的サイトとして安全にビルドし、Pages公開準備まで進める |

## 主要シナリオ

| ID | シナリオ | 成功条件 | 画面 |
|---|---|---|---|
| UC-01 | 初訪問者がトップで世界観を把握する | 大きな写真、短いコピー、控えめなCTAが上質に伝わる | `/` |
| UC-02 | Featured Collectionを見る | 代表作品3〜6点が余白を持って美しく見える | `/` |
| UC-03 | ブランドギャラリーで素材と作品の雰囲気を見る | 価格や購入導線なしで、作品名、素材、短いノートが分かる | `/`, `/works` |
| UC-04 | ブランド背景を読む | 手作り、素材へのこだわり、作り手の想いが短く上品に伝わる | `/about` |
| UC-05 | 所有者がローカルで確認し公開判断する | 表示確認・ビルド結果・未決定事項を見て承認可否を判断できる | docs, local preview |
