# Atelier Codia サイト仕様

## 目的

手作りアクセサリーの小さな個人ブランドサイトを作る。特定ブランドをコピーせず、白・アイボリー・淡いベージュ・ゴールド系の配色、余白、大きな写真、控えめなモーションで「小さな個人ブランドだけど、世界観は高級メゾンのように美しい」印象を目指す。

## スコープ

- ファーストビュー、ブランドストーリー、Featured Collection、素材ノート、About、404
- シルバー、パール、天然石、淡いゴールドを含む商品ビジュアル
- オンラインショップに誘導しないブランドギャラリー
- GitHub Pagesで公開できる build と手順
- レスポンシブ、代替テキスト、モーション軽減対応

## 非対象

- 決済、在庫管理、注文フォーム
- 商品価格表示、カート、購入導線
- 特定ブランドのロゴ・レイアウト・文言・装飾の模倣
- カスタムドメイン設定
- 法務判断の確定

## 完了の定義

- `/`, `/works`, `/about`, `/404` が端から端まで遷移できる
- スマホ/PCで高級感が崩れず、余白・写真・タイポグラフィが整っている
- 代表作品3〜6点と素材ノートが、価格表示なしで上品に表示される
- 生成画像が project 内に保存され、出所とプロンプトが記録されている
- `npm run check` が成功する
- GitHub Pages公開URLと検証結果が README と検証ログに残っている

## リスク・懸念

- 既存高級ジュエリーブランドの象徴的モチーフや見せ方に近づきすぎるリスク
- 生成画像が実物作品の写真だと誤認されるリスク
- 淡い配色でコントラストが不足するリスク
- GitHub Pages更新workflowが意図せず実行されるリスク

## テスト観点

- build が通る
- 主要ページと画像が dist に出力される
- ローカルリンクが壊れていない
- 画像altがある
- `prefers-reduced-motion` がCSSに含まれる
- PC/スマホでヒーロー、Collection、Aboutが崩れない
